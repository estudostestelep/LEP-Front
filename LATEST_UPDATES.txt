## Service Layer

All API communication happens through service layer:

```typescript
// User Service
userService.login(email, password)
userService.logout()
userService.getCurrentUser()
userService.getAll()
userService.getById(id)
userService.update(id, data)

// Product Service
productService.getAll()
productService.getById(id)
productService.create(data)
productService.update(id, data)
productService.delete(id)
productService.uploadImage(productId, formData)

// Reservation Service
reservationService.getAll()
reservationService.getById(id)
reservationService.create(data)
reservationService.update(id, data)
reservationService.cancel(id)
reservationService.getFiltered(filters)

// And 10+ more services...
```

## Error Handling

Axios interceptors handle all errors:

```typescript
// 401 - Unauthorized (token expired/invalid)
// -> Redirect to login, clear session

// 403 - Forbidden (insufficient permissions)
// -> Show permission error to user

// 404 - Not Found
// -> Show resource not found message

// 500 - Server Error
// -> Show server error message

// Network Error
// -> Show connection error with retry option
```
